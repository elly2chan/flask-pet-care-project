Project Overview

This project is an API service that provides various functionalities including user management, pet management, product catalog, and order placement.

Below you will find the details of the available API endpoints and how to interact with the system. You can register, log in, manage pets, manage products, and place orders using these endpoints.

The API uses JWT (JSON Web Tokens) for user authentication and authorization. The main components of the system are:

  • User Registration and Authentication
  • Pet Management (Add, Edit, Delete Pets)
  • Product Catalog (View, Add, Edit, Delete Products)
  • Order Management (Place Orders)

API Documentation

Authentication

POST /login - Login an existing user

POST /login
Content-Type: application/json
{
    "email": "user@example.com",
    "password": "your_password"
}

POST /register - Register a new user

POST /register
Content-Type: application/json
{
    "username": "newuser",
    "email": "newuser@example.com",
    "password": "your_password"
}

POST /users/change_password - Change your password

POST /users/change_password
Authorization: Bearer 
Content-Type: application/json
{
    "old_password": "oldpassword",
    "new_password": "newpassword"
}

Pet Management

POST /pets/add_pet - Add a new pet

POST /pets/add_pet
Authorization: Bearer 
Content-Type: application/json
{
    "name": "Fluffy",
    "species": "Cat",
    "age": 2
}

GET /pets - Retrieve all pets

GET /pets
Authorization: Bearer

Product Management

POST /products/add_product - Add a new product

POST /products/add_product
Authorization: Bearer 
Content-Type: application/json
{
    "name": "Pet Shampoo",
    "price": 10.99,
    "description": "Shampoo for your pets"
}

GET /products - Retrieve all products

GET /products

Order Management

POST /orders/place_order - Place an order

POST /orders/place_order
Authorization: Bearer 
Content-Type: application/json
{
    "product_id": 1,
    "quantity": 2
}
